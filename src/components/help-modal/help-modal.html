{{ currentLanguage === 'sk' ? 'Curry-Howardov izomorfizmus' : 'Curry-Howard isomorphism' }}

Curry-Howardov izomorfizmus je korešpondencia medzi formálnymi dôkazmi a programami. Logické výroky zodpovedajú typom a dôkazy zodpovedajú lambda termom daného typu. Ak existuje dôkaz výroku, existuje aj program (lambda term) žijúci v zodpovedajúcom type — a naopak.

The Curry-Howard isomorphism is a correspondence between formal proofs and programs. Logical propositions correspond to types, and proofs correspond to lambda terms of a given type. If a proof of a proposition exists, then a program (lambda term) inhabiting the corresponding type also exists — and vice versa.
